# Installation

## Requirements

- Spigot/Paper 1.21+
- Vault (required)
- PlaceholderAPI (optional)
- Java 17+ recommended

## Install Steps

1. Place `SimpleEconomy.jar` into `plugins/`.
2. Start the server once to generate configs.
3. Configure `plugins/SimpleEconomy/config.yml`.
4. (Optional) Configure `currencies.yml` for custom currencies.
5. Restart or run `/se reload`.

!!! warning
    If you select MySQL, change `database.password` from `CHANGEME` before starting.

!!! note
    Default language files are created under `languages/` on first run.
